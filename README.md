# to-do-list

### Welches Problem soll mit der Software gelöst werden?
Diese Software soll das Erledigen von Aufgaben im Alltag erleichtern.

### Features
**Aufgaben**
- erstellen
- bearbeiten
- löschen
- wiederherstellen

**Benachrichtigungen**
- Handy
- E-Mail
- Glocke oben?

**Zeit/Datum Anzeige**
- Uhrzeit
- Datums

**Daten sichern**
- Online Sichern
  - Account erstellen, einloggen
- Offline Sichern
  - Offline Backup speichern, wiederherstellen
- Daten
  - herunterladen
  - alle löschen
  - offline speichern

**Sprachen**
- Deutsch
- Englisch
